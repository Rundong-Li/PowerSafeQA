### 介绍

本项目基于 [InternLM 开源大模型](https://github.com/InternLM/InternLM)，在领域专有数据上进行指令微调，实现电力安全规程问答助手。欢迎大家也来参加书生大模型实战营项目！（http://github.com/internLM/tutorial）

### 功能

- 电力安全规程相关问题的智能问答
- 电力安全操作流程指导
- 电力设备使用注意事项查询
- 电力安全事故案例分析
- 电力安全法规政策解读

### 数据集

本项目使用了以下数据集进行训练:

- 电力安全事故案例库
- 电力设备操作手册

数据集经过脱敏处理，不包含敏感信息。

### 技术栈

基础模型: InternLM2.5-7B-Chat

微调方法: LoRA
